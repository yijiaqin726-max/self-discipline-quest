# 自律Quest —— 开发任务拆分

> 状态标记：⬜ 未开始 | 🔄 进行中 | ✅ 已完成

---

## Task 1：项目初始化 & 基础架构 ✅

**目标：** 搭建 Vue 3 项目骨架，配置路由、状态管理、全局样式

**子任务：**
- [ ] 用 Vite 创建 Vue 3 项目
- [ ] 安装依赖：vue-router、pinia
- [ ] 配置 Vue Router（5 个页面路由）
- [ ] 配置 Pinia store 目录结构
- [ ] 搭建侧边栏导航组件（AppSidebar）
- [ ] 配置全局 CSS 变量（颜色、字体、间距）
- [ ] 引入 Press Start 2P 字体（Google Fonts）
- [ ] 封装 useStorage composable（LocalStorage 读写）
- [ ] App.vue 基础布局（sidebar + router-view）

**产出：** 可在浏览器跑通的框架，5 个页面可跳转，侧边栏正常显示

---

## Task 2：个人主页 —— Profile 页面 ✅

**目标：** 实现 RPG 风格角色卡片页面

**子任务：**
- [ ] 创建 profileStore（昵称、座右铭、头像、总XP）
- [ ] 头像上传功能（读取本地图片转 base64 存储）
- [ ] 预设像素风头像选择（4-6 个内置选项）
- [ ] 昵称 / 座右铭行内编辑（点击进入编辑模式）
- [ ] 等级 & 称号计算逻辑（基于总XP自动展示）
- [ ] 技能熟练度总览进度条列表（从 skillStore 读取）
- [ ] 目标列表（增删改、勾选完成状态）
- [ ] Profile 页整体样式（RPG 卡片风格）

**产出：** 完整可交互的个人主页

---

## Task 3：打卡日历 —— Calendar 页面 ✅

**目标：** 实现当月可视化打卡日历

**子任务：**
- [ ] 创建 calendarStore（打卡记录，按 year-month 分组存储）
- [ ] 日历网格渲染（7列，正确计算月份第一天星期偏移）
- [ ] 打卡/取消打卡交互（点击 → 确认弹窗 → 切换状态）
- [ ] 打卡成功视觉效果（绿色发光 + CSS 动画）
- [ ] 清空本月按钮（二次确认弹窗）
- [ ] 统计栏：已打卡天数 / 当月总天数
- [ ] 连续打卡连击数计算与显示（🔥 连击）
- [ ] 打卡成功粒子特效（JS 实现）
- [ ] 日历格 hover 光效（CSS）

**产出：** 完整可交互的打卡日历页面

---

## Task 4：技能系统 —— Skills 页面 ✅

**目标：** 实现技能管理与熟练度进度展示

**子任务：**
- [ ] 创建 skillStore（技能列表、熟练度数值、已解锁奖励）
- [ ] 技能卡片组件（名称、进度条、当前值/下一里程碑）
- [ ] 添加新技能（弹窗输入名称）
- [ ] 删除技能（检查是否有关联任务，有则提示）
- [ ] 进度条动画（数值变化时平滑过渡）
- [ ] 里程碑检测逻辑（100 / 200 / 500 触发奖励弹窗）
- [ ] 奖励设置弹窗（用户输入自定义奖品文字并保存）
- [ ] 技能卡片显示"已解锁奖励"徽章
- [ ] 技能详情展开（历史里程碑奖励列表）

**产出：** 完整技能管理页面，里程碑奖励逻辑可运行

---

## Task 5：任务系统 —— Quest Board 页面 ✅

**目标：** 实现任务增删改查与 XP 记录联动

**子任务：**
- [ ] 创建 questStore（任务列表、完成次数）
- [ ] 任务表格组件（名称、XP、关联技能、完成次数、操作列）
- [ ] 新增任务表单（名称、XP下拉、技能选择/新建）
- [ ] `[+1完成]` 按钮逻辑：
  - 任务完成次数 +1
  - skillStore 对应技能熟练度 +XP
  - profileStore 总XP +XP
  - 触发里程碑检测
- [ ] XP 弹出动画（`+5 XP ⚡` 浮动消失效果）
- [ ] 编辑任务（行内编辑或弹窗）
- [ ] 删除任务（二次确认）
- [ ] 按技能分组视图 / 列表视图切换

**产出：** 完整任务看板，XP 与技能熟练度联动正常

---

## Task 6：番茄钟 —— Pomodoro 页面 ✅

**目标：** 实现番茄工作法计时器，完成后关联任务

**子任务：**
- [ ] 计时器核心逻辑（专注 25min / 休息 5min，setInterval）
- [ ] 自定义时长设置
- [ ] 开始 / 暂停 / 重置控件
- [ ] 圆形进度条可视化（SVG 或 CSS conic-gradient）
- [ ] 计时结束浏览器通知（Notification API，需用户授权）
- [ ] 完成弹窗："关联本次任务？" → 选择任务 → 记录 +XP
- [ ] 当日番茄计数统计

**产出：** 完整番茄钟页面，可与任务系统联动
